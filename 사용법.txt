ABAE 사용법
preprocess.py 실행시키면 데이터 만들어짐
- preprocessed_data 폴더에 만들어짐
preprocess.py 후에 word2vec.py 실행됨
- preprocessed_data 폴더에 embedding 파일 만들어짐
train.py 실행시켜서 학습시키면 됨
- aspect-size는 aspect를 몇개 추출할것인지
- emb-name은 embedding 파일 이름
- neg-size는 학습할 때 negative를 몇개 생성할지
- ortho-reg는 잘 모르겠음, lr  같은거인듯
- 그러면 output 폴더에 earphone 폴더에 결과 파일 나옴

aspect summary 사용법
construct.py 실행시키면 데이터 만들어줌
- data 폴더에 earphone 폴더에 train.json test.json train.source test.source 파일 만들어짐
- split : 데이터 파일 이름
finetune.py 실행시키면 학습이 됨
- dataset_name : 폴더 이름 넣어줌 (earphone 같은거)
get_model_binary.py 실행시켜서 체크포인트를 사용하여 바이너리 모델을 만듬
- hparams : 학습하면서 생성된 hyperparameter의 경로를 넣어줌 (lightening? log 폴더 안에 생성됨, yaml 형식)
- model_binary : checkpoint 파일 경로를 넣어줌
- output_dir : 모델 생성 할 경로를 넣어줌
generate.py 실행
- log_dir : 모델 경로 넣어줌
